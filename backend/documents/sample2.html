OQS-OpenSSL 3 Integration Guide

This document describes how OQS-OpenSSL 3 is integrated into this application to provide post-quantum secure TLS connections.

What is OQS-OpenSSL?

OQS-OpenSSL is a fork of OpenSSL that integrates post-quantum cryptographic algorithms from the Open Quantum Safe project. It allows applications to use post-quantum algorithms in TLS/SSL connections.

Architecture

Client → HTTPS (TLS 1.3 + PQ) → OQS-OpenSSL Proxy → FastAPI Backend

Key Features

  • Post-quantum key exchange using CRYSTALS-Kyber
  • Hybrid classical + post-quantum cipher suites
  • Backward compatibility with classical TLS
  • NIST-selected algorithms

Testing the Connection

To verify post-quantum TLS is working, use:

oqs_openssl s_client -connect localhost:443 -groups kyber512
